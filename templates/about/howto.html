Extracting features from a newly created corpus

This video shows how to create a graph for a new corpus.

There are 2 ways of exploring the main topics that have been extracted from the documents. The most obvious one, is to list all the features that have been created; the other way is to explore the graph that displays the connections between the documents that have been analysed and the features extracted from the materials.

Explore the graph

Create a new graph with 7 topics (features extracted from documents) and 10 words per topic

By default, every document in the graph has a maximal number of 3 features. This parameters can be modified, it is possible to create a graph with any maximal number of topics attached to a document. On the other hand, it is possible to create a list of features containing any number of articles, documents; the default number of documents per feature is set to 3.
